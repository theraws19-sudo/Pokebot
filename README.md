# Pokebot Project

## Структура проекта

- `main.py` — основной файл для запуска бота, обработки входящих сообщений и команд.
- `logic.py` — модуль с бизнес-логикой: классы, методы и утилиты для обработки данных.
- `config.py` — конфигурационный файл для хранения токена бота и других настроек.
